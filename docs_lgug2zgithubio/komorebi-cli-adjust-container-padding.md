```
Adjust container padding on the focused workspace

Usage: komorebic.exe adjust-container-padding <SIZING> <ADJUSTMENT>

Arguments:
  <SIZING>
          [possible values: increase, decrease]

  <ADJUSTMENT>
          Pixels to adjust by as an integer

Options:
  -h, --help
          Print help
```
