- `layout_center toggle`  toggles the layout on and off

<!--THE END-->

- `layout_center next`  switches to the next window (if layout is on) else runs the [next](#next) command

<!--THE END-->

- `layout_center prev`  switches to the previous window (if layout is on) else runs the [prev](#prev) command

<!--THE END-->

- `layout_center next2`  switches to the next window (if layout is on) else runs the [prev](#prev2) command

<!--THE END-->

- `layout_center prev2`  switches to the previous window (if layout is on) else runs the [prev2](#prev2) command
