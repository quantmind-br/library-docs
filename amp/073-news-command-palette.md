The Amp CLI and editor extensions now have something new: the Amp Command Palette.

You can open it via `Ctrl+O`/`Alt+O` in the CLI and `Cmd+Shift+A`/`Alt+Shift+A` in the editor extensions.

Unlike the old slash menu, the new command palette allows you to run commands without editing your prompt, and is accessible at all times (e.g. when the Amp editor panel isn't even open).

If you're on the CLI, this palette replaces the slash menu (and we'll pop open the new command palette if you type `/`).
