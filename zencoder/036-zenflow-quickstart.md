## Welcome to Zenflow

Zenflow transforms how you build software—from initial concept to production-ready code. This quickstart takes 5-10 minutes and shows you how to: \\

- Create a project and define your codebase scope \\
- Run AI agents to generate verified code \\
- Use workflows with built-in testing and validation \\
- Ship production-ready features faster

Zenflow organizes work into a hierarchy so every automation layer stays aligned. Explore each level below.

- Projects
- Tasks
- Subtasks & Chats
