All Mistral AI documentation: build, customize, and deploy AI your way.

## Products

Three products for different use cases:

| Product | Purpose | Link |
|---------|---------|------|
| **Le Chat** | Research, analyze documents, create agents — no code required | [chat.mistral.ai](https://chat.mistral.ai) |
| **Studio** | Build and deploy AI applications via API and playground | [console.mistral.ai](https://console.mistral.ai) |
| **Mistral Vibe** | Code, edit files, automate tasks from terminal | [docs](https://docs.mistral.ai/mistral-vibe/overview) |

## Developers

Build with Mistral models using the API, SDKs, and code examples.

- **API Reference**: endpoints, authentication, request/response schemas, error handling — [docs.mistral.ai/api](https://docs.mistral.ai/api)
- **SDKs**: Python, TypeScript, and other languages — [docs.mistral.ai/resources/sdks](https://docs.mistral.ai/resources/sdks)
- **Cookbooks**: step-by-step code walkthroughs for RAG, function calling, fine-tuning — [docs.mistral.ai/resources/cookbooks](https://docs.mistral.ai/resources/cookbooks)
- **Changelogs**: API updates, new model releases, deprecations — [docs.mistral.ai/resources/changelogs](https://docs.mistral.ai/resources/changelogs)

Start with a quickstart to send your first API request if you have not generated an API key yet.
