# focus

```
Change focus to the window in the specified direction

Usage: komorebic.exe focus <OPERATION_DIRECTION>

Arguments:
  <OPERATION_DIRECTION>
          [possible values: left, right, up, down]

Options:
  -h, --help
          Print help

```
