# ignore-rule

```
Add a rule to ignore the specified application

Usage: komorebic.exe ignore-rule <IDENTIFIER> <ID>

Arguments:
  <IDENTIFIER>
          [possible values: exe, class, title, path]

  <ID>
          Identifier as a string

Options:
  -h, --help
          Print help

```
