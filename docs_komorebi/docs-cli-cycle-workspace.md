# cycle-workspace

```
Focus the workspace in the given cycle direction

Usage: komorebic.exe cycle-workspace <CYCLE_DIRECTION>

Arguments:
  <CYCLE_DIRECTION>
          [possible values: previous, next]

Options:
  -h, --help
          Print help

```
