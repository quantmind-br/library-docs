## Homebrew on MacOS

[Section titled “Homebrew on MacOS”](#homebrew-on-macos)

Install using Homebrew:

Terminal window

```

brewinstallferoxbuster
```

## Homebrew on Linux via tap

[Section titled “Homebrew on Linux via tap”](#homebrew-on-linux-via-tap)

[Linux tap](https://github.com/TGotwig/homebrew-linux-feroxbuster/blob/main/feroxbuster.rb)

Terminal window

```

brewtaptgotwig/linux-feroxbuster
brewinstallferoxbuster
```
