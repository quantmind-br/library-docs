# swap-workspaces-with-monitor

```
Swap focused monitor workspaces with specified monitor

Usage: komorebic.exe swap-workspaces-with-monitor <TARGET>

Arguments:
  <TARGET>
          Target index (zero-indexed)

Options:
  -h, --help
          Print help

```
