# cycle-layout

```
Cycle between available layouts

Usage: komorebic.exe cycle-layout <CYCLE_DIRECTION>

Arguments:
  <CYCLE_DIRECTION>
          [possible values: previous, next]

Options:
  -h, --help
          Print help

```
