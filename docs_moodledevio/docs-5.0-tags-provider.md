## [AI Plugins](https://moodledev.io/docs/5.0/apis/plugintypes/ai)

The AI subsystem provides a way for developers to integrate external AI services into Moodle LMS.

## [AI Subsystem](https://moodledev.io/docs/5.0/apis/subsystems/ai)

The AI Subsystem in Moodle LMS provides a consistent and user-friendly way for users to interact with AI

## [Communication API](https://moodledev.io/docs/5.0/apis/subsystems/communication)

Communication API provides access to the messaging system and other communication providers (such as Matrix).

## [Communication plugin](https://moodledev.io/docs/5.0/apis/plugintypes/communication)

Communication plugin allows you to create a communication provider plugin, which can be added as a part of course or any other instances.

## [Providers](https://moodledev.io/docs/5.0/apis/plugintypes/ai/provider)

Providers are the interface between the AI subsystem and external AI.
