# manage-rule

```
Add a rule to always manage the specified application

Usage: komorebic.exe manage-rule <IDENTIFIER> <ID>

Arguments:
  <IDENTIFIER>
          [possible values: exe, class, title, path]

  <ID>
          Identifier as a string

Options:
  -h, --help
          Print help

```
