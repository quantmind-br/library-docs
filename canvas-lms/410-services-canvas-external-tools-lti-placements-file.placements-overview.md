Placements are the "entry points" into an External Tool. They determine where an External Tool is visible to users.

For placement settings, go to the [Tool Configuration](https://developerdocs.instructure.com/services/canvas/external-tools/lti/file.lti_dev_key_config#placements-params) page.

name: resource\_selection ⚠️

This is a placement that once was the default for 1.1 tools, and was one of the very first placements for Canvas. It makes the tool show up both in - [assignment\_selection](https://developerdocs.instructure.com/services/canvas/external-tools/lti/placements/file.placements_overview#assignment-selection) and [link\_selection](https://developerdocs.instructure.com/services/canvas/external-tools/lti/placements/file.placements_overview#link-selection) (new module item), but is deprecated now.

For 1.3 tools using this placement we highly recommend to use assignment\_selection or link\_selection instead! You can read about those in details [here](https://developerdocs.instructure.com/services/canvas/external-tools/lti/placements/file.assignment_selection_placement) and [here](https://developerdocs.instructure.com/services/canvas/external-tools/lti/placements/file.link_selection_placement).
