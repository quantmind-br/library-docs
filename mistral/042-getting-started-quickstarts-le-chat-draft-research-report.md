Draft a short research report using three core tools in [Le Chat](https://chat.mistral.ai).

**Tools used:**
- **Web Search**: pull in current data from the internet
- **Document Analysis**: upload a file and extract key points
- **Canvas**: refine the output in a side-by-side editor

**Time to complete:** ~10 minutes

**Prerequisites:**
- Mistral AI account (Free plan supports Web Search and Document Uploads). [Create account](https://chat.mistral.ai)
- Optional: document to analyze (PDF, Word, or text format)

## Steps

### 1. Web Search

1. Open [New chat](https://chat.mistral.ai).
2. Select **Web Search** icon (globe) under the input bar.
3. Ask about a recent event:

> What are the three most significant AI industry news announcements from the past 7 days? Summarize each in one sentence.

### 2. Document Analysis

1. In the same chat, click the **paperclip icon** to upload a document (or provide a URL).
2. Ask Le Chat to extract insights:

> Based on the context in this document, how does it relate to the news announcements you just found? Explain the connection step-by-step.

Le Chat reads the document and synthesizes the answer, keeping context from your previous search.

### 3. Canvas

1. Ask Le Chat to generate a draft:

> Draft a short memo based on the document and the news events.

2. Click **Open in Canvas** button near the response.
3. In the Canvas view on the right, highlight a sentence you want to change.
4. An inline prompt appears. Type *"Make this sentence more professional"* or *"Expand on this idea"*.

## Outcome

You successfully used three of Le Chat's core capabilities:
1. **Web Search** for pulling in current data
2. **Document Analysis** for extracting key points from uploaded files
3. **Canvas** for interactive, side-by-side editing
