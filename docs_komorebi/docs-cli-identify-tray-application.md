# identify-tray-application

```
Identify an application that closes to the system tray

Usage: komorebic.exe identify-tray-application <IDENTIFIER> <ID>

Arguments:
  <IDENTIFIER>
          [possible values: exe, class, title, path]

  <ID>
          Identifier as a string

Options:
  -h, --help
          Print help

```
