# resize-delta

```
Set the resize delta (used by resize-edge and resize-axis)

Usage: komorebic.exe resize-delta <PIXELS>

Arguments:
  <PIXELS>
          The delta of pixels by which to increase or decrease window dimensions when resizing

Options:
  -h, --help
          Print help

```
