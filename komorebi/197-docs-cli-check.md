# check

```
Check komorebi configuration and related files for common errors

Usage: komorebic.exe check [OPTIONS]

Options:
  -k, --komorebi-config <KOMOREBI_CONFIG>
          Path to a static configuration JSON file

  -h, --help
          Print help

```
