# send-to-named-workspace

```
Send the focused window to the specified workspace

Usage: komorebic.exe send-to-named-workspace <WORKSPACE>

Arguments:
  <WORKSPACE>
          Target workspace name

Options:
  -h, --help
          Print help

```
