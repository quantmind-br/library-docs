All our documentation in your hands: build, customize, and deploy AI, your way.

## Products

Mistral AI has four interfaces:

| Interface | Purpose | Audience |
|---|---|---|
| **Le Chat** | AI workspace for chat, search, document analysis, and agents | Business users |
| **Studio** | Developer console for API keys, model testing, fine-tuning, evaluation | Developers |
| **Mistral Vibe** | Terminal-native coding agent powered by Devstral | Developers |
| **Admin** | Control plane for organization setup, billing, SSO, team management | IT/Security |

## Le Chat

Use Le Chat's collaborative AI workspace. No code required.

[Open Le Chat](https://chat.mistral.ai)

## Studio

Build, test, and deploy AI applications with the Mistral API.

[Open Studio](https://console.mistral.ai)

## Mistral Vibe

Generate code, edit files, and automate tasks from your terminal. Integrates with JetBrains, Zed, and any IDE supporting the Agent Communication Protocol. Can run fully offline with local models.

[Try Mistral Vibe](https://github.com/mistralai/mistral-vibe)

## Admin

Control plane for IT and security teams. Manage organization account, team structure, billing, SSO, and security policies.

[Open Admin](https://admin.mistral.ai)

## Developers

Build with Mistral models using the API, SDKs, and code examples.

Start with a quickstart to send your first API request, then explore:
- [**API Reference**](https://docs.mistral.ai/api) — endpoints, authentication, request/response schemas, error handling
- [**SDKs**](https://docs.mistral.ai/resources/sdks) — Python, TypeScript, and other languages
- [**Cookbooks**](https://docs.mistral.ai/resources/cookbooks) — RAG, function calling, fine-tuning, integration patterns
- [**Changelogs**](https://docs.mistral.ai/resources/changelogs) — API updates, new models, deprecations

## Admin (continued)

The Admin Panel controls access, enforces security policies, and monitors usage across workspaces. Choose a quickstart to begin.
