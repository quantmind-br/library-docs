Warning:

Commercial Hardware tools and the eSDK are available only for approved partners

Info:

You need to get explicit approval before you start your integration in order to pass certification for smart displays. If you are using a Systems Integrator pleases reach out to them directly.

A device qualifies as a smart display if it features a touch screen and voice. If you are uncertain whether your device qualifies as a smart display, you can reach out to our [service desk](https://spotify-service-desk.atlassian.net/servicedesk/customer/portal/3).
