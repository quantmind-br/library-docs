Le Chat's built-in knowledge comes from data collected up to a certain date (its *knowledge cutoff*), so it may not know about very recent events.

Web search bridges that gap by letting Le Chat browse the internet in real time, so you get **up-to-date answers backed by verifiable sources**.

## How to Enable

1. Click the `+` icon or type `/` in the chat window.
2. Select `Tools` then enable `Web search`.

## How It Works

When you ask a question that needs current information, Le Chat **searches the web** and weaves the findings into its response. Ask naturally.

When Le Chat uses web search, you'll notice:
- A **globe icon** next to the response, confirming web data was used
- **Inline links** pointing back to the original sources
- A `Sources` button at the bottom of the response that opens a panel listing every reference

Every **web-sourced response comes with references** so you can check the underlying material.

## News Sources

For news queries, Le Chat draws from professional news partners:
- [Agence France-Presse (AFP)](https://www.afp.com/en)
- [Associated Press (AP)](https://apnews.com/)

Results come from **verified, editorially vetted sources** you can cite with confidence.

When news results are included, you'll see a **news icon** alongside the globe icon.

## Best Use

For quick competitive or market intelligence:
- *"Latest regulatory updates in [your industry]"*
- *"Recent funding rounds in [sector]"*

## When to Use Other Tools

| Tool | Use Case |
|------|----------|
| **Web search** | Targeted, single-step lookups for current information |
| [**Deep Research**](https://docs.mistral.ai/le-chat/research-analysis/deep-research) | Comprehensive, multi-source report with structured citations |
| [**Open URL**](https://docs.mistral.ai/le-chat/research-analysis/open-url) | Analyze a specific web page by pasting its URL |
| [**Code Interpreter**](https://docs.mistral.ai/le-chat/content-creation/code-interpreter) | Run Python on data you've gathered from the web |
