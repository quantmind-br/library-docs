# focused-workspace-container-padding

```
Set container padding on the focused workspace

Usage: komorebic.exe focused-workspace-container-padding <SIZE>

Arguments:
  <SIZE>
          Pixels size to set as an integer

Options:
  -h, --help
          Print help

```
