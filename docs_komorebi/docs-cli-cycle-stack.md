# cycle-stack

```
Cycle the focused stack in the specified cycle direction

Usage: komorebic.exe cycle-stack <CYCLE_DIRECTION>

Arguments:
  <CYCLE_DIRECTION>
          [possible values: previous, next]

Options:
  -h, --help
          Print help

```
