The `lms link set-device-name` command sets a display name for this device, visible to other devices on the link.

## Rename this device[](#rename-this-device "Link to 'Rename this device'")

```

lms link set-device-name "My Mac Studio"
```

The new name takes effect immediately and is visible to connected peers via [`lms link status`](https://lmstudio.ai/docs/cli/link/link-status).
