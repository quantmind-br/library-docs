### [hashtag](#available-commands) Available Commands

[**dap snapshot**](https://developerdocs.instructure.com/services/dap/dap-cli-readme/dap-cli-reference/dap-cli-reference-snapshot) Download complete snapshot of tables.

[**dap incremental**](https://developerdocs.instructure.com/services/dap/dap-cli-readme/dap-cli-reference/dap-cli-reference-incremental) Download changes from tables within a specified time range.

[**dap list**](https://developerdocs.instructure.com/services/dap/dap-cli-readme/dap-cli-reference/dap-cli-reference-list) Get the list of all tables available in a namespace.

[**dap schema**](https://developerdocs.instructure.com/services/dap/dap-cli-readme/dap-cli-reference/dap-cli-reference-schema) Download the schema of tables.

[**dap initdb**](https://developerdocs.instructure.com/services/dap/dap-cli-readme/dap-cli-reference/dap-cli-reference-initdb) Import a full snapshot of tables into a database.

[**dap syncdb**](https://developerdocs.instructure.com/services/dap/dap-cli-readme/dap-cli-reference/dap-cli-reference-syncdb) Synchronize table data changes to a database.

[**dap dropdb**](https://developerdocs.instructure.com/services/dap/dap-cli-readme/dap-cli-reference/dap-cli-reference-dropdb) Delete synchronized tables in a database.

[**dap env vars**](https://developerdocs.instructure.com/services/dap/dap-cli-readme/dap-cli-reference/dap-cli-reference-env-vars) Available enviroment variables.
