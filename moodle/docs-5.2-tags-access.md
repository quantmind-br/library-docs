## [Access API](https://moodledev.io/docs/5.2/apis/subsystems/access)

The Access API gives you functions so you can determine what the current user is allowed to do. It also allows plugins to extend Moodle with new capabilities.

## [Quiz access rule sub-plugins](https://moodledev.io/docs/5.2/apis/plugintypes/quizaccess)

Quiz access rule sub-plugins extend the ability to add conditions a user must meet to attempt a given quiz.
