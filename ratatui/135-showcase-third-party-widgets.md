To add your widgets to this list, please read the [Third Party Widgets README](https://github.com/ratatui/ratatui-website/blob/main/code/showcase/widget-showcase-third-party/README.md)

[`ratatui-image`](https://crates.io/crates/ratatui-image) is an image widget with multiple graphics protocol backends (sixel, iTerm2, kitty, etc).

![ratatui-image demo](https://github.com/benjajaja/ratatui-image/blob/master/assets/Screenshot.png?raw=true)

[`throbber-widgets-tui`](https://crates.io/crates/throbber-widgets-tui) is a widget that displays throbber. Also-known-as an activity indicator, progress bar, loading icon, spinner, or ぐるぐる (guru guru).

![throbber-widgets-tui](https://ratatui.rs/_astro/throbber-widgets-tui.D4P-gN-G_A6KhC.webp)

[`tui-big-text`](https://crates.io/crates/tui-big-text) is a rust crate that renders large pixel text as a widget using the glyphs from the [font8x8](https://crates.io/crates/font8x8) crate.

![tui-big-text](https://ratatui.rs/_astro/tui-big-text.CUuhZyfA_2gBgcv.webp)

[`tui-checkbox`](https://crates.io/crates/tui-checkbox) is a customizable checkbox widget with support for custom styling, symbols (unicode, emoji, ASCII), and optional block wrappers.

![tui-checkbox](https://ratatui.rs/_astro/tui-checkbox.UlicjJ0z_hxuUf.webp)

[`tui-logger`](https://crates.io/crates/tui-logger) is widget for capturing and displaying logs.

![tui-logger](https://ratatui.rs/_astro/tui-logger.BETHNVBY_1W9FLH.webp)

[`tui-menu`](https://crates.io/crates/tui-menu) is a menu widget that renders nestable menus.

![tui-menu](https://ratatui.rs/_astro/tui-menu.CZ2DQEbw_1ipy3l.webp)

[`tui-nodes`](https://crates.io/crates/tui-nodes) is a Node graph visualization widget.

![tui-nodes](https://ratatui.rs/_astro/tui-nodes.Dbivca9__gqvkW.webp)

[`tui-piechart`](https://crates.io/crates/tui-piechart) is a versatile piechart widget that comes in both a standard and high resolution with a large set of predefined symbols and the ability to add custom ones.

![tui-piechart](https://ratatui.rs/_astro/tui-piechart.BUCvPlsw_Z2tAgTr.webp)

[`tui-scrollview`](https://crates.io/crates/tui-scrollview) is a widget for creating scrollable views.

![Demo of tui-scrollview](https://vhs.charm.sh/vhs-6PuT3pdwSTp4aTvKrCBx9F.gif)

[`tui-term`](https://crates.io/crates/tui-term) is a pseudoterminal widget.

![Demo of tui-term](https://vhs.charm.sh/vhs-4zK1zlTOSueAmlOkZlssBr.gif)

[`tui-textarea`](https://crates.io/crates/tui-textarea) is a simple yet powerful text editor widget like `<textarea>` in HTML for ratatui and tui-rs. Multi-line text editor can be easily put as part of your TUI application.

![tui-textarea](https://ratatui.rs/_astro/tui-textarea.B7Kkoa7u_2qMr7w.webp)

[`tui-tree-widget`](https://crates.io/crates/tui-tree-widget) is a widget built to show Tree Data structures.

![tui-tree-widget](https://ratatui.rs/_astro/tui-treeview.CqZqDoUO_1XYvTg.webp)

[`tui-widget-list`](https://crates.io/crates/tui-widget-list) is stateful widget list implementation `List` for Ratatui that allows to work with any list of widgets that implement to the `Listable` trait.

![tui-widget-list](https://ratatui.rs/_astro/tui-widget-list.VjkkypPB_SJOS2.webp)
