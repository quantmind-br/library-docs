## Portal de integração OLX

- Olá nobre pessoa desenvolvedora, este é o portal de integração da OLX.
- Aqui você encontrará informações sobre as diversas formas de integrar seu produto programaticamente com os nossos sistemas.
- [Comece aqui](https://developers.olx.com.br/anuncio/home.html) se você tem interesse nas nossas integrações para anúncios.
- [Por aqui](https://developers.olx.com.br/chat/home.html) para integração de chat.
- [Ou por aqui](https://developers.olx.com.br/lead/home.html) para integração de leads.
- [Ou por aqui](https://developers.olx.com.br/destaque/home.html) para aplicar destaques e consulta do plano.
- [Ou por aqui](https://developers.olx.com.br/webhooks/home.html) para configurar Webhooks de notificações.
- [Ou por aqui](https://developers.olx.com.br/anuncio/api/published_ads_status.html) para consultar status atual do anúncio.

Usuários da API, estamos implementando novas políticas de rate-limit. Para obter informações detalhadas, consulte [aqui](https://developers.olx.com.br/faq/rate_limit.html).
