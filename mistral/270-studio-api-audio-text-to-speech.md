# Text to Speech

**Voxtral TTS** is Mistral's text-to-speech model with zero-shot voice cloning. Generates natural, expressive speech from text using a short audio prompt.

## Key Features

| Feature | Description |
|---------|-------------|
| **Zero-shot voice cloning** | Clone any voice from 2–3 seconds of audio, capturing emotion, speaking style, and accent |
| **Voice-as-an-instruction** | Model follows intonation, rhythm, and emotional rendering without prosody/emotion tags |
| **Multilingual** | English, French, Spanish, Portuguese, Italian, Dutch, German, Hindi, Arabic. Supports cross-lingual voice cloning and code-mixing |
| **Streaming** | ~90ms processing time. End-to-end API time-to-first-audio varies by format (~0.8s for `pcm`, ~3s for `mp3`) |
