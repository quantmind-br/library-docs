# move

```
Move the focused window in the specified direction

Usage: komorebic.exe move <OPERATION_DIRECTION>

Arguments:
  <OPERATION_DIRECTION>
          [possible values: left, right, up, down]

Options:
  -h, --help
          Print help

```
