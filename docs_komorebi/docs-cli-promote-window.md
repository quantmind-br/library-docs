# promote-window

```
Promote the window in the specified direction

Usage: komorebic.exe promote-window <OPERATION_DIRECTION>

Arguments:
  <OPERATION_DIRECTION>
          [possible values: left, right, up, down]

Options:
  -h, --help
          Print help

```
