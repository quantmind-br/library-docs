# transparency

```
Enable or disable transparency for unfocused windows

Usage: komorebic.exe transparency <BOOLEAN_STATE>

Arguments:
  <BOOLEAN_STATE>
          [possible values: enable, disable]

Options:
  -h, --help
          Print help

```
