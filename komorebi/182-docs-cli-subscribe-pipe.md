# subscribe-pipe

```
Subscribe to komorebi events using a Named Pipe

Usage: komorebic.exe subscribe-pipe <NAMED_PIPE>

Arguments:
  <NAMED_PIPE>
          Name of the pipe to send event notifications to (without "\\.\pipe\" prepended)

Options:
  -h, --help
          Print help

```
