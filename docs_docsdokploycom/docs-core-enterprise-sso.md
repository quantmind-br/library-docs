Configure SSO with Auth0, Keycloak, or other OIDC/SAML providers

Enterprise supports Single Sign-On via OpenID Connect (OIDC) and SAML. You can use Auth0, Keycloak, or any compatible identity provider.

Choose a provider below for step-by-step configuration:

- [**Auth0**](https://docs.dokploy.com/docs/core/enterprise/sso/auth0) — Cloud identity platform
- [**Azure AD (Microsoft Entra ID)**](https://docs.dokploy.com/docs/core/enterprise/sso/azure) — Microsoft's cloud identity platform
- [**Okta**](https://docs.dokploy.com/docs/core/enterprise/sso/okta) — Cloud identity platform
- [**Keycloak**](https://docs.dokploy.com/docs/core/enterprise/sso/keycloak) — Open-source identity and access management

You can also use any other OIDC/SAML provider by configuring the endpoints and flow manually.

For other OIDC/SAML providers, use the same endpoints and flow; [contact us](https://dokploy.com/contact) if you need help.
