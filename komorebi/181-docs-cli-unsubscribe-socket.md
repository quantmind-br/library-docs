# unsubscribe-socket

```
Unsubscribe from komorebi events

Usage: komorebic.exe unsubscribe-socket <SOCKET>

Arguments:
  <SOCKET>
          Name of the socket to stop sending event notifications to

Options:
  -h, --help
          Print help

```
