Le Chat is Mistral's **conversational AI assistant**, available at [chat.mistral.ai](https://chat.mistral.ai), on [iOS](https://apps.apple.com/us/app/le-chat-by-mistral-ai/id6740410176), and on [Android](https://play.google.com/store/apps/details?id=ai.mistral.chat).

Instead of juggling separate tools for writing, research, coding, or image generation, Le Chat lets you do it all in one conversation.

## Conversation Modes

| Mode | Description |
|------|-------------|
| [**Chat**](https://docs.mistral.ai/le-chat/conversation/chat) | Multi-turn conversations with Mistral models |
| [**Think mode**](https://docs.mistral.ai/le-chat/conversation/think-mode) | Step-by-step reasoning for complex problems (chain-of-thought) |
| [**Voice mode**](https://docs.mistral.ai/le-chat/conversation/voice-mode) | Speak instead of type; supports multiple languages |
| [**Flash Answers**](https://docs.mistral.ai/le-chat/conversation/flash-answers) | Near-instant responses for quick queries |

## Capabilities

- **Content creation**: draft reports, proposals, emails; translate between languages
- **Data analysis**: work through formulas, financial models, or data questions
- **Code**: generate, troubleshoot, and review code with best-practice suggestions
- **Research**: get explanations, competitive analysis, or quick summaries

Open [Le Chat](https://chat.mistral.ai) to start (no account required). Sign in to save conversations and access all features.

Available on several plans from Free to Enterprise. See [pricing page](https://mistral.ai/pricing) for details.
