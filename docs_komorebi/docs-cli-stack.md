# stack

```
Stack the focused window in the specified direction

Usage: komorebic.exe stack <OPERATION_DIRECTION>

Arguments:
  <OPERATION_DIRECTION>
          [possible values: left, right, up, down]

Options:
  -h, --help
          Print help

```
