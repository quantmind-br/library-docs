Firecrawl は、[AGPL-3.0 ライセンス](https://github.com/firecrawl/firecrawl/blob/main/LICENSE) の下で公開されているオープンソースです。最高のプロダクトを提供するために、追加機能とマネージドインフラを備えたホスティング型クラウド版も [firecrawl.dev](https://firecrawl.dev) で提供しています。

## 機能比較

オープンソース版には、コアとなるスクレイピングエンジンとクロールエンジンが含まれます。Firecrawl Cloud では、オープンソース版のすべてに加えて、インフラストラクチャーレベルの機能、管理ダッシュボード、エンタープライズ向け機能が追加されています。 ![Firecrawl Cloud とオープンソースの比較](https://mintcdn.com/firecrawl/vlKm1oZYK3oSRVTM/images/open-source-cloud.png?fit=max&auto=format&n=vlKm1oZYK3oSRVTM&q=85&s=763a6e92c8605d06294ed7ed45df85d0)

## なぜ両方あるのか？

クラウドソリューションにより、すべてのユーザーに対して継続的な改良と高品質で持続可能なサービスを実現します。Firecrawl Cloud の収益は、オープンソースプロジェクトの継続的な開発に充てられます。

- **オープンソース** — Firecrawl を自分のインフラ上で実行し、コアのスクレイピングエンジンを完全に制御できます。[Running locally](https://docs.firecrawl.dev/ja/contributing/guide) および [Self-hosting](https://docs.firecrawl.dev/ja/contributing/self-host) を参照してください。
- **クラウド** — 強化されたプロキシ、利用状況ダッシュボード、エンタープライズ機能を備えたマネージドサービスで、すぐに利用を開始できます。[firecrawl.dev](https://firecrawl.dev) からサインアップしてください。
