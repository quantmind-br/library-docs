# move-to-named-workspace

```
Move the focused window to the specified workspace

Usage: komorebic.exe move-to-named-workspace <WORKSPACE>

Arguments:
  <WORKSPACE>
          Target workspace name

Options:
  -h, --help
          Print help

```
