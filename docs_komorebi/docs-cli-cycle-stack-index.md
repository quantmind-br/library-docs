# cycle-stack-index

```
Cycle the index of the focused window in the focused stack in the specified cycle direction

Usage: komorebic.exe cycle-stack-index <CYCLE_DIRECTION>

Arguments:
  <CYCLE_DIRECTION>
          [possible values: previous, next]

Options:
  -h, --help
          Print help

```
