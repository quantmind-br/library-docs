# mouse-follows-focus

```
Enable or disable mouse follows focus on all workspaces

Usage: komorebic.exe mouse-follows-focus <BOOLEAN_STATE>

Arguments:
  <BOOLEAN_STATE>
          [possible values: enable, disable]

Options:
  -h, --help
          Print help

```
