# flip-layout

```
Flip the layout on the focused workspace

Usage: komorebic.exe flip-layout <AXIS>

Arguments:
  <AXIS>
          [possible values: horizontal, vertical, horizontal-and-vertical]

Options:
  -h, --help
          Print help

```
