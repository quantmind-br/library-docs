# clear-named-workspace-rules

```
Remove all application association rules for a named workspace

Usage: komorebic.exe clear-named-workspace-rules <WORKSPACE>

Arguments:
  <WORKSPACE>
          Name of a workspace

Options:
  -h, --help
          Print help

```
