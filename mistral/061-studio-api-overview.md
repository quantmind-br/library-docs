[Studio](https://console.mistral.ai) is Mistral's developer platform for AI applications. Use the API to build conversational AI, autonomous agents, document intelligence, and RAG applications. Use the console to manage reusable building blocks, test models, and monitor usage.

## Quick Start

1. **Create an API key** in the console or through the [API keys](https://docs.mistral.ai/admin/security-access/api-keys) page in Admin.
2. **Send your first request**: follow the [send your first API request](https://docs.mistral.ai/getting-started/quickstarts/developer/first-api-request) quickstart to make a chat completion call in Python or TypeScript.

Explore [chat completions](https://docs.mistral.ai/studio-api/conversations/chat-completion) guide, then other features based on your requirements.

## Playground

Open [Studio](https://console.mistral.ai) → **Playground** to test prompts, compare models, and experiment with parameters — no code needed. From there, manage API keys, review usage, and set up agents.

> [!info]
> Some features may vary depending on your plan. Check the [pricing page](https://mistral.ai/pricing) for details.
