## Svelte 5 vs Svelte 4 Event Handling

**Key improvements in Svelte 5:**

Svelte 4 Svelte 5 Benefit `on:click={fn}` `onclick={fn}` Standard HTML attributes `createEventDispatcher` + `e.detail` Callback props with direct values Zero boilerplate, better types `on:click|preventDefault|once` `once(preventDefault(fn))` Composable JS functions `<button on:click>` (forwarding) `{...props}` spreading Explicit and flexible
