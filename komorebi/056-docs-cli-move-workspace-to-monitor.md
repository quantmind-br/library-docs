# move-workspace-to-monitor

```
Move the focused workspace to the specified monitor

Usage: komorebic.exe move-workspace-to-monitor <TARGET>

Arguments:
  <TARGET>
          Target index (zero-indexed)

Options:
  -h, --help
          Print help

```
