# focused-workspace-padding

```
Set workspace padding on the focused workspace

Usage: komorebic.exe focused-workspace-padding <SIZE>

Arguments:
  <SIZE>
          Pixels size to set as an integer

Options:
  -h, --help
          Print help

```
