# stackbar-mode

```
Set the stackbar mode

Usage: komorebic.exe stackbar-mode <MODE>

Arguments:
  <MODE>
          Desired stackbar mode
          
          [possible values: always, never, on-stack]

Options:
  -h, --help
          Print help

```
