Ratatui provides [templates](https://github.com/ratatui/templates) for building apps using [Cargo Generate](https://cargo-generate.github.io/cargo-generate/).

To use these, [install Cargo Generate](https://cargo-generate.github.io/cargo-generate/installation.html) and then create a new app using the chosen template:

Terminal window

```

cargoinstallcargo-generate
cargogenerateratatui/templates
```
