| **Studio** | Activate API, generate keys, test models in playground |
| **Mistral Vibe** | Install CLI, generate code, scaffold projects from terminal |
| **Developer** | Set up SDK, send API requests, build agents |
| **Admin** | Create organization, configure SSO, manage workspaces |
