Code with Qwen Code

Qwen Code is an open-source AI agent for the terminal, optimized for Qwen series models. It helps you understand large codebases, automate tedious work, and ship faster.

&gt;\_`bash -c "$(curl -fsSL https://qwen-code-assets.oss-cn-hangzhou.aliyuncs.com/installation/install-qwen.sh)" -s --source qwenchat`

Multi-protocol, OAuth free tier

Use OpenAI / Anthropic / Gemini-compatible APIs, or sign in with Qwen OAuth for 1,000 free requests/day.

Open-source, co-evolving

Both the framework and the Qwen series models are open-source—and they ship and evolve together.

Agentic workflow, feature-rich

Rich built-in tools (Skills, SubAgents) for a full agentic workflow and a Claude Code-like experience.

![Terminal-first, IDE-friendly](https://img.alicdn.com/imgextra/i3/O1CN01fmGlRx23mXc5gd9Ed_!!6000000007298-2-tps-2068-1482.png)

Terminal-first, IDE-friendly

Built for developers who live in the command line, with optional integration for VS Code, Zed, and JetBrains IDEs.

![Interactive Mode](https://img.alicdn.com/imgextra/i2/O1CN01QTcJCI22DReBdld1d_!!6000000007086-2-tps-2068-1482.png)

Interactive Mode

Run 'qwen' in your project folder to launch the interactive terminal UI. Use '@' to reference local files and get contextual assistance for your codebase.

`qwen`

![Headless Mode](https://img.alicdn.com/imgextra/i1/O1CN017S4VTQ1nTH6GH2WMq_!!6000000005090-2-tps-2068-1482.png)

Headless Mode

Use '-p' flag to run Qwen Code without the interactive UI — ideal for scripts, automation, and CI/CD pipelines. Integrate AI into your existing workflows seamlessly.

`qwen -p "Explain the auth flow"`

![IDE Integration](https://img.alicdn.com/imgextra/i2/O1CN01JYWqkT1bbj1DFMBdl_!!6000000003484-2-tps-2652-1428.png)

IDE Integration

Use Qwen Code inside your favorite editor — VS Code, Zed, and JetBrains IDEs are all supported. Get AI-powered coding assistance without leaving your development environment.

`Search "Qwen Code" in VS Code`

![Developer SDK](https://img.alicdn.com/imgextra/i1/O1CN017JZGMg1XvBNihQlUn_!!6000000002985-2-tps-1540-1124.png)

Developer SDK

Build your own AI agents and tools on top of Qwen Code. Our SDK provides programmable access to the same powerful engine that drives the CLI.

`npm install @qwen-code/sdk`
