## Installation

Mistral Vibe is a Python package installable as a command-line tool.

**Perfect for:**
- Developers working **terminal-first**
- Anyone wanting an AI assistant that interacts with their **local environment**
- Teams needing reproducible, script-friendly workflows in the **command line**

## Quick Install

```bash
pip install mistral-vibe
```

Or via `uv`:

```bash
uvx mistral-vibe
```

> [!note]
> Ensure Python 3.12+ is installed. If you encounter issues with `uv`, try using the `--no-cache` flag.

Mistral Vibe works on Windows, but officially supports and targets UNIX environments.

## Verify Installation

```bash
mistral-vibe --version
```
