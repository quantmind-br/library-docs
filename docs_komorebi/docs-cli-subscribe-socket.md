# subscribe-socket

```
Subscribe to komorebi events using a Unix Domain Socket

Usage: komorebic.exe subscribe-socket <SOCKET>

Arguments:
  <SOCKET>
          Name of the socket to send event notifications to

Options:
  -h, --help
          Print help

```
