# cycle-send-to-workspace

```
Send the focused window to the workspace in the given cycle direction

Usage: komorebic.exe cycle-send-to-workspace <CYCLE_DIRECTION>

Arguments:
  <CYCLE_DIRECTION>
          [possible values: previous, next]

Options:
  -h, --help
          Print help

```
